Больше не нужно тратить уйму времени, чтобы добавить товары “Натали” на ваш сайт, ведь теперь в один клик вы сможете синхронизировать весь каталог магазина с вашим ресурсом, да еще и абсолютно бесплатно! И это возможно благодаря нашему модулю, который позволяет поддерживать актуальную информацию для пользователей вашей площадки, уделяя этому минимум времени. Хотите синхронизировать более 6000 товаров всего за 5 минут? Сделайте это вместе с нами!

== Description ==

Внимание! Данный модуль не удаляет и не изменяет ранее добавленные вами товары.

Для корректной работы плагина необходима пара RestApi ключей WooCommerce с правами уровня Чтение/запись.


== Первоначальный импорт ==

Шаг 1.
На первом шаге нужно заполнить пару RestApi ключей WooCommerce.
Выбрать тип цены Натали для импорта и указать наценку в процентах.
В третьем блоке выбрать необходимые разделы для импорта и связать их с разделами, которые есть на вашем сайте

Шаг 2.
Перед импортом происходит сбор информации о товарах из выбранных категорий. В случае выбора большого количества разделов эта операция
может занять до 10 минут. После завершения этой операции появится информация о количестве разделов и товаров котрые будут импортированы на сайт.
Импорт будет запущен после нажатия на кнопку "Начать импорт".
В любой момент вы можете нажать "Пауза" и посмотреть, в отдельной вкладке как происходит импорт товара в каталоге.

Внимание! Повторная настройка удалит старые настройки, и для корректной работы обновлений потребуется заново провести весь импорт.

== Обновления ==

Модуль актуализирует каталог Natali. Снятые с продажи элементы удаляются, новые создаются, а измененные - изменяются.

После настройки и импорта товаров зайдите в "Обновление каталога" и нажмите на кнопку
"Получить данные". После этого будут загружены все актуальные на сегодняшний день изменения по товарам.

Далее нажмите "обновить элементы" и модуль пройдется по обновленным элементам. Обновления для разделов которые не привязаны будут пропущены.
Для привязанных разделов могут создаваться новые товары.

== Требования к хостингу ==
Кроме стандартных get и post запросов, для работы плагина небоходима поддержка put и delete запросов к сайту. Такие запросы не дожны блокироваться хостером либо другими плагинами.

== Автоматическое обновление ==
Для автоматического обновления товаров необходимо поставить задание в cron 
php /путь_к_папке_сайта/wp-content/plugins/cadesign-natali/updateNatali.php example.com
wp-content/plugins - адрес папки с плагинами. Если она у Вас отличается, то укажите свой адрес.
